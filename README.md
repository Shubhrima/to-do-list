 <h1>Bucket List</h1>

A simple web app that allows people to make multiple to do lists. One can add, and remove items from the lists.

<strong>Technologies/Frameworks</strong>
<ul>
 <li>Bootstrap</li>
 <li>Node.js</li>
 <li>MongoDB Atlas</li>
 </ul>
